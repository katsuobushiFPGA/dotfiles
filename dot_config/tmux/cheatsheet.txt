 tmux cheatsheet
 操作: Ctrl+b を離してから次のキー
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 [ペイン]
  Ctrl+b → |    縦に分割
  Ctrl+b → -    横に分割
  Ctrl+b → p    ペイン番号で移動
  Ctrl+b → z    ズーム切り替え
  Ctrl+b → x    ペインを閉じる

 [ウィンドウ]
  Ctrl+b → c    新しいウィンドウ
  Ctrl+b → w    番号で移動
  Ctrl+b → n    次のウィンドウ
  Ctrl+b → ,    名前を変更

 [セッション]
  Ctrl+b → d    デタッチ
  Ctrl+b → $    名前を変更

 [その他]
  Ctrl+b → r    設定リロード
  Ctrl+b → [    コピーモード
  Ctrl+b → ?    キー一覧
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 mouse ON (クリック/スクロール)
