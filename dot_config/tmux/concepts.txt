
 tmux 概念
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 [Server]
  tmux本体のプロセス。
  最初のセッション作成時に自動起動し、
  全セッションがなくなると終了する。

 [Session]
  tmuxの作業単位。
  サーバ上に複数持てる。
  デタッチしても生き続ける。
  → 例: 「作業A用」「作業B用」

 [Window]
  セッション内のタブのようなもの。
  1セッションに複数持てる。
  → 例: 「編集用」「サーバ用」

 [Pane]
  ウィンドウを分割した画面。
  1ウィンドウに複数持てる。
  → 例: 左にエディタ、右にログ

 [構造まとめ]
  Server
  └─ Session (複数可)
     └─ Window (複数可)
        └─ Pane (複数可)
