============================================================
  nvim チートシート  (Leader = Space)
============================================================

[ ファイル操作 ]
  <leader>w       保存
  <leader>q       閉じる
  <leader>t       ファイルツリー 開閉

[ ウィンドウ移動 ]
  Ctrl+h/j/k/l    左/下/上/右のウィンドウへ移動

[ Telescope ]
  <leader>ff      ファイル名で検索
  <leader>fg      ファイル内容をgrep
  <leader>fb      バッファ一覧

[ LSP ]
  gd              定義へジャンプ
  gr              参照一覧
  K               ドキュメント表示
  <leader>rn      リネーム
  <leader>ca      コードアクション
  [d / ]d         前/次のエラーへ

[ Git ]
  ]g / [g         次/前の変更箇所へ
  <leader>gp      変更箇所をプレビュー
  <leader>gr      変更箇所をリセット
  <leader>gg      LazyGit を開く

[ AI (Claude Code) ]
  <leader>ac      Claude Code 開閉
  <leader>as      選択範囲を Claude に送信 (visual)

[ 編集 ]
  gc              コメントアウト/解除
  < / >           インデント (選択維持)

============================================================
