============================================================
  nvim チートシート  (Leader = Space)
============================================================

[ ファイル操作 ]
  <leader>w       保存
  <leader>q       閉じる
  <leader>t       ファイルツリー 開閉

[ ウィンドウ移動 ]
  Ctrl+h/j/k/l    左/下/上/右のウィンドウへ移動

[ NvimTree: 基本 ]
  <leader>t       ツリー 開閉
  g?             ヘルプ表示 (このキー一覧をツリー内で確認)
  q              ツリーを閉じる
  R              ツリーを再描画
  <C-k>          ファイル情報を表示

[ NvimTree: 移動 ]
  j / k          下 / 上へ移動
  <CR> / o       ファイルを開く / ディレクトリ展開
  <Tab>          プレビュー (フォーカスは残す)
  P              親ディレクトリへ
  <BS>           親ディレクトリを閉じる
  -              一段上のディレクトリへ (CD)
  <C-]>          ディレクトリへ CD (ルート変更)
  K / J          兄弟の最初 / 最後へ
  < / >          前 / 次の兄弟へ
  W              全ディレクトリを畳む
  E              全ディレクトリを展開

[ NvimTree: ファイルを開く ]
  <C-v>          縦分割で開く
  <C-x>          横分割で開く
  <C-t>          新規タブで開く
  s              システム既定アプリで開く

[ NvimTree: ファイル操作 ]
  a              作成 (末尾に / でディレクトリ)
  d              削除
  D              ゴミ箱へ
  r              リネーム
  <C-r>          フルパスでリネーム
  e              ベース名のみリネーム
  x              切り取り
  c              コピー
  p              貼り付け
  y              ファイル名をコピー
  Y              相対パスをコピー
  gy             絶対パスをコピー

[ NvimTree: 表示・フィルタ ]
  H              隠しファイル表示切替 (dotfiles)
  I              gitignore 表示切替
  U              カスタム除外の表示切替
  f              フィルタ入力
  F              フィルタ解除
  S              ファイル検索

[ NvimTree: マーク・Git ]
  m              ブックマーク切替
  bmv            ブックマーク済みファイルを移動
  ]c / [c        次 / 前の Git 変更
  ]e / [e        次 / 前の診断 (LSP)

[ Telescope ]
  <leader>ff      ファイル名で検索
  <leader>fg      ファイル内容をgrep
  <leader>fb      バッファ一覧

[ LSP ]
  gd              定義へジャンプ
  gr              参照一覧
  K               ドキュメント表示
  <leader>rn      リネーム
  <leader>ca      コードアクション
  [d / ]d         前/次のエラーへ

[ Git ]
  ]g / [g         次/前の変更箇所へ
  <leader>gp      変更箇所をプレビュー
  <leader>gr      変更箇所をリセット
  <leader>gg      LazyGit を開く

[ AI (Claude Code) ]
  <leader>ac      Claude Code 開閉
  <leader>as      選択範囲を Claude に送信 (visual)

[ 編集 ]
  gc              コメントアウト/解除
  < / >           インデント (選択維持)
